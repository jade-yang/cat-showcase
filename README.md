# 猫咪图鉴 - 超级互动版

一个充满趣味的猫咪展示网页，包含丰富的互动功能、数据可视化和趣味游戏。

## 功能特点

### 基础功能
- 12只可爱的猫咪卡片展示
- 支持按性格标签筛选（粘人、活泼、安静、爱睡觉、胆小、好奇、贪吃）
- 搜索猫咪名字或品种
- 多种排序方式（名字、年龄、点赞数、热度）
- 用户上传新猫咪

### 互动功能
- 点赞、收藏、分享猫咪
- 点击卡片查看详细信息
- 连续点击触发彩蛋动画
- 悬浮毛线球游戏（拖动毛线球增加分数）
- 连击系统
- 随机推荐猫咪按钮

### KPI 数据系统
- 完整的热度分数计算（likes×2 + favorites×3 + shares×4 + views×0.5 + gameInteractions×1.5）
- 5级成长系统（休息区萌猫 → 宇宙猫王）
- 视图追踪（IntersectionObserver）
- 实时数据面板（Chart.js 图表）
- TOP 10 排行榜

### 数据可视化
- 点赞排行榜 TOP 10 柱状图
- 收藏排行榜 TOP 10 柱状图
- KPI 分布饼图
- 热度趋势折线图

### 随机事件系统
- 每 20-40 秒随机触发事件
- 自动给猫咪增加浏览量

## 技术栈

- HTML5 + CSS3
- JavaScript (原生)
- Chart.js 4.4.0
- localStorage 数据持久化
- IntersectionObserver API

## 预览

直接在浏览器中打开 `index.html` 即可预览。

或者使用本地服务器：
```bash
npx serve .
```
